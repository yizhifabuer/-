# -纯文字恶魔轮盘赌
===========================================思维导图===========================================  
恶魔轮盘赌 (恶魔轮盘赌.py)
│
├──  导入模块
│   ├── random      → 随机数生成（子弹、道具、血量）
│   └── os          → 系统操作（预留）
│
├──  常量定义
│   └── ITEM_NAMES  → 道具英文→中文映射字典
│       ├── "phone"            → "手机"
│       ├── "cigarette"        → "香烟"
│       ├── "expired_medicine" → "过期药"
│       ├── "handcuffs"        → "手铐"
│       ├── "knife"            → "刀"
│       ├── "beer"             → "啤酒"
│       ├── "magnifier"        → "放大镜"
│       ├── "adrenaline"       → "肾上腺素"
│       └── "inverter"         → "转换器"
│
└── 🏛️ 类: InteractiveTextGame
    │
    ├──  __init__(self, player)
    │   ├── 玩家数据: {"name":"", "hp":随机2-5, "max_hp":同上}
    │   ├── 恶魔数据: {"name":"恶魔", "hp":同上, "max_hp":同上}
    │   ├── 道具列表: all_items (9种)
    │   ├── 背包: player_inventory[], devil_inventory[]
    │   ├── 枪膛: chamber[], current_index=0
    │   ├──  状态标记
    │   │   ├── knife_active            → 刀是否激活
    │   │   ├── handcuffs_used_this_round → 本回合手铐是否用过
    │   │   ├── player_skip             → 玩家被铐跳过
    │   │   ├── devil_skip              → 恶魔被铐跳过
    │   │   ├── current_round_shots[]   → 本回合已知子弹记录
    │   │   └── _new_round_player_first → 新回合玩家先手标记
    │   └── 回合计数: round_count=0
    │
    ├── 道具辅助方法
    │   ├── _get_inventory(user)        → 根据user返回对应背包
    │   ├── _remove_item(inv, item)     → 从背包移除道具
    │   └── _count_item(inv, item)      → 统计道具数量
    │
    ├── 枪械系统
    │   ├── _load_chamber()
    │   │   ├── 随机2-8发子弹，一半真一半假
    │   │   ├── 洗牌打乱
    │   │   ├── 重置 current_index=0
    │   │   ├── 清空 current_round_shots
    │   │   ├── 双方各抽2-4个道具
    │   │   ├── 重置所有状态标记
    │   │   └── 设置 _new_round_player_first = True ★
    │   │
    │   ├── _show_round_info()          → 打印回合信息（真/假子弹数）
    │   │
    │   ├── fire()
    │   │   ├── 子弹打完? → _load_chamber() + 返回None ★
    │   │   ├── 取出当前子弹
    │   │   ├── current_index++
    │   │   ├── knife_active = False    → 刀效果只持续一发
    │   │   └── 返回子弹类型("real"/"fake")
    │   │
    │   └── _attack_target(target, attacker_name)
    │       ├── 调用 fire() 获取子弹
    │       ├── 子弹=None? → 返回 (False, False, True) ★
    │       ├── 计算伤害: damage = 2 if knife_active else 1 ★
    │       ├── 实弹 → 扣血, 刀失效
    │       ├── 空包弹 → 刀失效
    │       ├── 空包弹打自己 → 返回 (死亡?, True, False) 保持回合
    │       └── 返回 (目标死亡?, 保持回合?, 子弹打完?)
    │
    ├── 道具使用方法 (9种)
    │   ├── use_phone(user)
    │   │   └── 随机查看剩余子弹中某一发 → 记录到 current_round_shots
    │   │
    │   ├── use_cigarette(user)
    │   │   └── 回复1HP（不超过max_hp）
    │   │
    │   ├── use_expired_medicine(user)
    │   │   └── 50%回2HP / 50%扣1HP
    │   │
    │   ├── use_handcuffs(user) ★修改
    │   │   ├── 每回合限一次
    │   │   ├── 玩家用 → devil_skip = True （铐恶魔）
    │   │   └── 恶魔用 → player_skip = True （铐玩家）
    │   │
    │   ├── use_knife(user) ★修改
    │   │   ├── 不能重复激活
    │   │   └── knife_active = True → 下一发伤害翻倍
    │   │
    │   ├── use_beer(user) ★修改
    │   │   ├── current_index < len(chamber)? → current_index++ (跳过当前子弹)
    │   │   └── 否则: 枪膛已空
    │   │
    │   ├── use_magnifier(user)
    │   │   └── 查看当前子弹 → 记录到 current_round_shots
    │   │
    │   ├── use_adrenaline(user)
    │   │   └── 偷对手一个非肾上腺素道具
    │   │
    │   └── use_inverter(user)
    │       └── 反转当前子弹（真↔假）
    │
    ├── 恶魔AI系统
    │   ├── _devil_use_item(item_name)
    │   │   └── 打印使用信息 + 调用对应道具方法
    │   │
    │   ├── _devil_calculate_real_probability()
    │   │   ├── 基于已知信息（current_round_shots）计算
    │   │   └── 返回剩余实弹概率
    │   │
    │   └── enemy_turn() ★修改
    │       ├── 检查 _new_round_player_first → 跳过，玩家先手 ★
    │       ├── 检查 devil_skip → 被铐跳过 ★
    │       ├── 阶段1: 狂用香烟+过期药（回血）
    │       ├── 阶段2: 肾上腺素偷对手道具
    │       ├── 阶段3: 手铐直接铐玩家
    │       ├── 阶段4: 计算实弹概率（不输出）★
    │       │   ├── prob > 0.5 → 打玩家，有刀先用刀
    │       │   └── prob ≤ 0.5 → 有转换器? 反转后打玩家 : 打自己
    │       ├── 执行 _attack_target()
    │       ├── 子弹打完? → 不显示状态，返回continue ★
    │       ├── 目标死亡? → 返回 win/lose
    │       ├── 保持回合? → 递归调用 enemy_turn()
    │       └── 否则显示状态 → 返回 continue
    │
    ├── 玩家回合系统
    │   ├── _show_inventory(inv)         → 格式化显示道具栏
    │   │
    │   ├── _show_status() ★修改
    │   │   ├── 双方HP
    │   │   ├── 双方道具
    │   │   ├── 剩余子弹数
    │   │   └── 刀激活状态
    │   │
    │   ├── combat_round(user, enemy) ★修改
    │   │   ├── 检查 player_skip → 被铐跳过 ★
    │   │   ├── 显示状态
    │   │   ├── 菜单: ①打对手 ②打自己 ③用道具 ④逃跑
    │   │   ├── 选项1/2 → _attack_target()
    │   │   │   ├── 子弹打完? → 返回 continue
    │   │   │   ├── 显示状态
    │   │   │   ├── 死亡? → win/lose
    │   │   │   └── 保持回合? → 递归
    │   │   ├── 选项3 → _player_use_item()
    │   │   └── 选项4 → flee
    │   │
    │   └── _player_use_item(user, enemy)
    │       ├── 显示道具列表（中文名+数量）
    │       ├── 玩家选择道具
    │       ├── 调用对应道具方法
    │       └── 成功后返回 combat_round（继续用道具或开枪）
    │
    └── 主循环
        ├── run()
        │   ├── 输入角色名
        │   ├── 显示初始血量
        │   ├── 外层循环（每轮）
        │   │   ├── round_count++
        │   │   ├── _load_chamber() + _show_round_info()
        │   │   ├── 内层循环（回合内）
        │   │   │   ├── 重置 _new_round_player_first = False ★
        │   │   │   ├── combat_round() → 玩家回合
        │   │   │   │   └── 返回 win/lose/flee → _end_game()
        │   │   │   └── enemy_turn() → 恶魔回合
        │   │   │       └── 返回 win/lose → _end_game()
        │   │   └── 恶魔死亡? → 复活 + 询问继续
        │   └── 退出
        │
        └── _end_game(result)
            ├── win  → 恭喜玩家
            ├── lose → 玩家被击败
            └── flee → 玩家逃跑

           2.很多ai新创建的变量和函数，只是正常的用法，但是我还是想不到，比如 _devil_calculate_real_probability，对恶魔计数只有模糊的认识，看过结果之后其实不复杂，但归根结底问题的关键还是自己想出来
